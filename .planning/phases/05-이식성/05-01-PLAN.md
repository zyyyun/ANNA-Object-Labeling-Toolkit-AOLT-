    ```xml
        <AssemblyVersion>1.0.0.0</AssemblyVersion>
        <FileVersion>1.0.0.0</FileVersion>
        <InformationalVersion>1.0.0</InformationalVersion>
        <Company>ANNA</Company>
    ```

    ```
    bin/
    obj/
    .claude/
    installer/Output/
    publish/
    *.user
    ```

    3. Create empty file `installer/.gitkeep` so the `installer/` directory is tracked. Plan 02 will add `ASLT-Setup.iss` and `README.md` here.

    No code changes to C# source files.
